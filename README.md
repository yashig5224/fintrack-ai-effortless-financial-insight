# 🌿 TrackMint

TrackMint is an AI-powered personal finance management platform that helps users track expenses, manage budgets, monitor subscriptions, and gain actionable financial insights. With smart analytics, automated categorization, and personalized recommendations, TrackMint makes money management simple, secure, and efficient.

## 🚀 Key Features

* 💰 Expense & Income Tracking
* 📊 Spending Analytics & Reports
* 🤖 AI-Powered Financial Insights
* 📅 Budget Planning & Alerts
* 💳 Subscription Management
* 🔔 Smart Notifications & Reminders
* 🔐 Secure Authentication
* ☁️ Cloud-Based Data Synchronization

## 🛠 Tech Stack

* **Frontend:** React, TypeScript, Tailwind CSS, Vite
* **Backend:** Supabase, PostgreSQL, Edge Functions
* **Authentication:** Supabase Auth
* **Payments:** Razorpay
* **Deployment:** Vercel / Netlify

## 🎯 Vision

TrackMint empowers users to take control of their finances through intelligent tracking, data-driven insights, and proactive financial planning.
